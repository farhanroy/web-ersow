Our Divisions

Our team is divided into several divisions that have their own tasks.

Mechanic

Make a robot body shape and everything that a robot needs in terms of physics

Computer Vision

Utilizing Machine Learning to detect objects around the robot through the camera.

Embedded System

Programming the microcontroller so that it can process data from the sensor

Non Technical

In charge of managing non-technical things like Instagram Post

Artificial Intelligence

Programming a robot to make a decision using AI.

Communication

Create a program that connects AI and Vision along with other supporting software
